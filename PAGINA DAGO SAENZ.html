RESUMEN BÁSICO DE HTML

INTRODUCCIÓN

A continuacion realizaremos un breve resumen acerca de como elaborar una pagina por medio de HTML partiendo de herramientas basicas y usando como editor BRACKETS que es ideal para este tipo de trabajo.

COMANDOS

  1. Salto de linea
    • < br/ >
  2. Párrafo
    • < p >
  3. Titulos
    • <hi>,<h2>,<h3>,<h3>,<h4>, <h5>,<h6>
  4. Énfasis
    • <em> <strong>
  5. Hipervinculo a otra página del mismo sitio
    • <a href="MULTIMEDIA\HTML></a>
  6. hipervinculo a otro sitio de internet
    • < a href=” http://www.maestrodelacomputacion.net/”> Maestros de la Computación</a>
  7. Imágenes dentro de una pagina
    • <img src=”Foto.jpg” alt=”Pintura”>
  8. Hipervínculo mediante una imagen
    • <a href=” http://www.maestrodelacomputacion.net/”><img src=”Foto.jpgalt=”Logo”</a>
  9. Apertura de un hipervínculo en otra instancia del navegador
    • <a href=” http://www.maestrodelacomputacion.net/” target”_blank”> maestro de la computación</a>
  10. Lista ordenada
    • <ol>
  11. Lista no ordenada
    • <ul>
